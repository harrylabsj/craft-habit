# Craft Habit
为音乐、绘画、写作等创意技能设计可持续的每日练习习惯，让坚持不再靠意志力。

## What it does
- 🎨 个性化练习蓝图：提供最低版、标准版、挑战版三档方案
- 🔗 习惯绑定：将新练习锚定到已有习惯上（如“喝完咖啡后练音阶”）
- 🛡️ 障碍应对手册：针对完美主义、疲惫、无聊等常见卡点的 if-then 策略

## Example scenarios

**1. 想培养每天读书的习惯**
> 👤 想养成每天读书半小时的习惯，但坚持了3天就放弃了，有什么方法能坚持下去？
> 🤖 习惯养成四步法：1）降低门槛——从每天读5分钟开始，确保'不可能失败'；2）习惯绑定——'每天刷完牙后读5分钟'；3）环境设计——把书放在床头而非手机旁；4）追踪记录——用Notion/小日常App记录连续天数。建议从微信读书开始利用通勤零碎时间。

**2. 设计每日钢琴练习习惯**
> 👤 I want to practice piano daily but only have 10 minutes available.
> 🤖 三版本方案：最低版（2分钟手指热身）、标准版（10分钟热身+一个乐段）、挑战版（20分钟完整练习）。习惯绑定："喝完早晨咖啡后弹热身音阶"。包含热身与收尾仪式。

**3. 克服写作完美主义**
> 👤 I keep rewriting my first paragraph and never finish anything I write.
> 🤖 障碍应对手册——"如果完美主义发作，用5分钟不限质量的自由写作练习，不允许编辑。"包含7天启动计划，重点是产出数量而非质量。
